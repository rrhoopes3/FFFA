FFFA Sprite Sheet Example

Unit: Alley Tabby Thug (Tank)

Sprite Sheet Layout: Each row represents one animation. Each column is a frame within that animation. All frames are a uniform 64×64 px cell size. The full sheet below is 384×448 px (6 columns × 7 rows).
alley_tabby_thug_sheet.png — 384 × 448

Sheet Layout Map

Animation Previews

Usage in code:
To render frame n of animation row r:

sx = n * 64;   sy = r * 64;
ctx.drawImage(sheet, sx, sy, 64, 64, destX, destY, 64, 64);

Recommended frame rates:
Idle: 4 fps • Walk: 8 fps • Attack: 12 fps • Hurt/Death: 10 fps • Block: 6 fps
